How-To Add Your Theme

  • Fork the site (e.g. drjekyllthemes.github.io) on GitHub
  • Add a new entry in the _data/themes.yml file and fill out all fields. Example: 
    - title:     Shiori
  homepage:  http://ellekasai.github.io/shiori/
  download:  http://github.com/ellekasai/shiori/
  demo:      http://ellekasai.github.io/shiori/
  author:    Elle Kasai
  thumbnail: shiori.png
  license:   MIT License
  • Make a 250 x 200 thumbnail and drop it in the thumbnails folder. Note: Do NOT forget to list its filename in the themes.yml entry.
  • Test it out, then push your changes up and open a pull request.
  • Thanks!
